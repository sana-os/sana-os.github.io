5. Core Principleの不変性

SANA OS Core Principle（core/SANA_OS_Core_Principle.md）は、フレームワークの基本的な不変量を定義しています。いかなる派生物においても改変することはできません。

すべての派生物において保持しなければならない4つの不変量：（1）born_loved = true（非敵対的基準線）、（2）観察なき論理は存在しない、（3）構造的翻訳、（4）自律性の保全。

5.1 目的の保全

SANA OSの根本的な目的—「結論に急ぐ前に前提を整えることで、対話における構造的摩擦を軽減する」—は、いかなる改変においても維持されなければなりません。

SANA OS素材を操作・強制・欺瞞的な出力の生成を目的として転用することは禁止されています。
